5 aplicaciones WEB mas famosas de la actualidad

Facebook

Fecha de lanzamiento: 2004
Empresa: Facebook
Caracteritica mas novedosa: Facebook ofrece una amplia variedad de aplicaciones y juegos que los usuarios pueden jugar y utilizar para personalizar su experiencia en la red social.

  • Thrift: Un kit de herramientas para la interoperabilidad de lenguajes de programación, que permite combinar el uso de lenguajes como C++, Java, Python, PHP, Ruby, Haskell, y más.
  • LAMP: Un ecosistema que incluye Linux, Apache, MySQL y PHP, que se utiliza para administrar los recursos de cada servidor, almacenar datos de forma estructurada y automatizar la creación de páginas web.
  • Scribe: Un sistema de distribución y registro de mensajes en los servidores, tolerante a fallas y capaz de funcionar sobre una gran cantidad de equipos.
  • HipHop for PHP: Un transformador de código PHP en código C++, que optimiza el desempeño del lenguaje PHP.
  • Cassandra: Una base de datos NoSQL que se utiliza para almacenar los datos de los usuarios de forma estructurada y distribuida.

Tiktok

Fecha de lanzamiento: Septiemmbre 2016
Empresa: ByteDance
Caracteristica mas novedosa: TikTok cuenta con una gran biblioteca de música que los usuarios pueden utilizar como fondo para sus videos.

  • Java para el servidor principal)
  • Go para algunos servicios
  • MySQL para la base de datos)
  • React Native para la interfaz de usuario
  • JavaScript para el lado del cliente

Instagram

Fecha de lanzamiento: 2010
Empresa: Kevin Systrom y Mike Krieger
Caracteristica mas novedosa: nterfaz centrada en imágenes y videos: Instagram se enfoca en la publicación y visualización de contenido multimedia, como fotos y videos, en lugar de texto o enlaces.
Tenologias que se utilizaron para su desarrollo

  • WSGI (Web Server Gateway Interface): Instagram utiliza Gunicorn como servidor WSGI en lugar de mod_wsgi y Apache, debido a su mayor facilidad de configuración y menor intensidad en la CPU.
  • Fabric: Se utiliza para desplegar ejecuciones en paralelo y rápido, permitiendo a Instagram escalarse rápidamente.
  • Django: La aplicación web de Instagram se construyó utilizando Django, un framework de Python.
  • Amazon EC2: Instagram utiliza instancias de Amazon EC2 para ejecutar su aplicación, lo que les permite escalarse según las necesidades de demanda.
  • Ubuntu Linux 11.04: Las instancias de EC2 de Instagram utilizan Ubuntu Linux 11.04.

Youtube

Fecha de lanzamiento: Febrero 2005
Empresa: Chad Hurley, Steve Chen y Jawed Karim, tres ex empleados de PayPal.
Caracteristica mas novedosa: cualquier persona puede subir vídeos en YouTube, con una duración máxima de 12 horas.
Tenologias que se utilizaron para su desarrollo

  • Amazon EC2: Servicio de computación en la nube de Amazon.
  • Apache Web Server: Servidor web de código abierto..
  • COREhub Domain Registration: Registro de dominios..
  • AComodo SSL: Certificado de seguridad SSL.
  • Cowboy Web Server: Servidor web de código abierto..

Snapchat


Fecha de lanzamiento: 2011
Empresa: Evan Spiegel, Bobby Murphy y Reggie Brown.
Caracteristica mas novedosa: Filtros: Snapchat ofrece una variedad de filtros de realidad aumentada y efectos visuales que pueden ser aplicados a las historias.
Tenologias que se utilizaron para su desarrollo

  • Realidad Aumentada (RA): Snapchat ha desarrollado tecnologías de RA para crear experiencias inmersivas, como los Lentes de RA, que permiten a los usuarios agregar efectos y filtros a sus fotos y videos.
  • Reconocimiento Facial: Snapchat ha trabajado en tecnologías de reconocimiento facial para identificar y procesar imágenes de rostros, lo que permite funciones como la autenticación de usuarios y la personalización de contenido.
  • Machine Learning (ML): Snapchat utiliza ML para mejorar la precisión de sus algoritmos de recomendación de contenido y para desarrollar funciones como la detección de objetos y la segmentación de imágenes..
  • Computer Vision: La visión por computadora se utiliza en Snapchat para analizar y procesar imágenes y videos, permitiendo funciones como la detección de objetos, la segmentación de imágenes y la aplicación de efectos visuales.
